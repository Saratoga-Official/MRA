## 功能

1. 开始前任务

2. 自动远征

3. 自动战役

4. 常规图（1-9章）的自动出征

5. 自动演习

6. 无消耗刷1-1

7. 自动决战E6（NEW）

8. 结束前任务

9. 在新活动出之后更新活动练级
